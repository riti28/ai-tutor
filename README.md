AI Tutor Web Application

This project is an AI-powered tutoring system where students can:

• Select grade
• Select subject
• Select textbook (NCERT / ICSE / Stateboard)
• Select language

The AI answers questions based on the selected textbook content.

Features:
- Textbook-based AI answers
- Multilingual support
- Voice explanation
- FastAPI backend
- Gemini AI integration

Technology Stack:
Frontend: HTML, CSS, JavaScript
Backend: FastAPI (Python)
AI Model: Google Gemini
Speech: gTTS
